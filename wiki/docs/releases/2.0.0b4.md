# CaeReflex `2.0.0b4`

Gate 7 adds the deterministic physics-consistency protocol `caereflex.physics-rule/1.0` and the first versioned OpenFOAM CFD pack, `caereflex.openfoam-cfd/1.0.0`.

The initial pack checks declared dimensions for `U`, `p` and `nu`, internal face accounting, boundary patch ranges and field-class associations. Every result carries an explicit status, evidence pointers, missing evidence, assumptions, remediation and limitations.

Missing or malformed evidence never becomes a pass. Internal evaluation failures fail closed as `blocked`.

This release does not assess convergence, mesh independence, turbulence-model suitability, numerical accuracy, experimental agreement, certification or design safety.
