# CaeReflex `2.0.0b3`

Gate 6C adds bounded spatial queries and freezes Gate 6 acceptance.

## Added

- `caereflex.spatial-query/1.0` read-only query results.
- Graph, frame, entity, relation, neighbourhood, bounds and ArrayRef-link queries.
- Configurable page, offset, scan, traversal and serialised-response ceilings.
- Same-frame bounds queries with no implicit transform or unit conversion.
- Recorded-relation traversal with no inferred adjacency.
- `caereflex.gate6.spatial/1.0` snapshot/store/query acceptance reports.
- Canonical snapshot SHA-256 digests and shared ArrayRef-registry checks.
- `caereflex spatial` CLI commands.
- Gate 6 freeze CI across Python 3.10–3.12.

## Compatibility

ReflexCase remains `1.0`; the inspection contract remains `2.0-alpha.3`; the spatial graph remains `1.0`; and Gate 6B mapping remains `caereflex.spatial-mapping/1.0`.

## Safety boundary

Acceptance establishes contract and persistence compatibility only. This release does not assert cross-format equivalence, compose coordinate frames, infer topology, validate mesh quality or physics, or certify engineering safety.
