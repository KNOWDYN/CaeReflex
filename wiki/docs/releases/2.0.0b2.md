# CaeReflex 2.0.0b2

`2.0.0b2` implements Gate 6B by mapping frozen native-reader evidence from OpenFOAM, Gmsh and VTK into the canonical spatial graph introduced in Gate 6A.

## Version alignment

- Package: `2.0.0b2`
- ReflexCase schema: `1.0`
- Inspection contract: `2.0-alpha.3`
- Spatial graph format: `1.0`

## Mapping coverage

- OpenFOAM: mesh region, declared patches, field dataset block and lazy arrays.
- Gmsh: source blocks, geometry entities, physical groups and lazy arrays.
- VTK: decoded dataset blocks, bounds and point/cell/field arrays.

The mapping is deterministic and backend-neutral. It preserves source-native identifiers and provenance without asserting cross-format equivalence.

## Coordinate-frame posture

Backend coordinates are attached to unresolved source frames. CaeReflex does not assume metres, a zero origin, global Cartesian axes, right-handed orientation or a transform between files.

## Safety boundary

This release does not validate simulations. It does not infer adjacency, assess mesh quality, prove convergence, validate physics, certify engineering results or establish design safety.

## Deferred

Gate 6C will add bounded spatial queries and acceptance fixtures for equivalent geometry, transformed frames, mixed cells, disconnected/non-manifold meshes, and unresolved units or names.
